Name

mt_widget_get_parent — returns parent of a widget

Synopsis

MT_WIDGET *mt_widget_get_parent(MT_WIDGET *widget, MT_WIDGET *result);

Description

Returns the parent widget or NULL when the widget don't have a parent. If you don't pass the result, the result will be newly allocated, you should then release the memory using the free function.
